# Personal Knowledge Management System (PKMS)

A Senior-level JavaScript project demonstrating:

- ES Modules, Classes, and OOP
- EventBus and Plugin System
- Undo/Redo (Command Pattern)
- State Management (Redux-like)
- LocalStorage-ready architecture

## Features
- Create and manage notes
- Tagging and versioning
- Plugin support (e.g., word count)
- Event-driven architecture

## How to run
1. Open `index.html` with Live Server or Python HTTP server
2. Check console for notes and events
